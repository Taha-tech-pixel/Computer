Octal Number System (Base-8)

The base-8 system using digits 0-7, historically important in computing

Base-8 Historical Unix Permissions Digital Systems

Overview

What is Octal?

Octal is a base-8 number system that uses eight digits: 0, 1, 2, 3, 4, 5, 6, and 7. It was historically popular in computing because early computers often used word sizes that were multiples of 3 bits, making octal a natural fit for representing binary data.

Historical Significance

Octal was widely used in early computing systems, particularly with PDP and other minicomputers. While less common today, it still appears in Unix file permissions, some programming contexts, and digital electronics.

Modern Usage

Today, octal is primarily used in Unix/Linux file permissions (chmod), some assembly languages, and specialized applications where base-8 representation is advantageous for grouping binary digits.

Octal Fundamentals

Octal Digits and Binary Relationship

Decimal | Octal | Binary
   0    |   0   |  000
   1    |   1   |  001
   2    |   2   |  010
   3    |   3   |  011
   4    |   4   |  100
   5    |   5   |  101
   6    |   6   |  110
   7    |   7   |  111

Each octal digit represents exactly 3 binary bits

Unix File Permissions

Permission Structure

Permission Bits:
r w x | r w x | r w x
4 2 1 | 4 2 1 | 4 2 1
Owner | Group | Others

Example: chmod 755 file
7 = 4+2+1 = rwx (owner: read, write, execute)
5 = 4+0+1 = r-x (group: read, execute)
5 = 4+0+1 = r-x (others: read, execute)

Common Permission Combinations

755 (rwxr-xr-x)

Executable files, scripts

644 (rw-r--r--)

Regular files, documents

600 (rw-------)

Private files (owner only)

777 (rwxrwxrwx)

Full permissions (rarely used)

Programming with Octal

JavaScript Implementation

// Convert decimal to octal
function decimalToOctal(num) {
    return num.toString(8);
}

// Convert octal to decimal
function octalToDecimal(octalStr) {
    return parseInt(octalStr, 8);
}

// File permission decoder
function decodePermissions(octal) {
    const permissions = ['---', '--x', '-w-', '-wx', 'r--', 'r-x', 'rw-', 'rwx'];
    const digits = octal.toString().split('');
    
    return digits.map(digit => permissions[parseInt(digit)]).join(' ');
}

// Examples
console.log(decimalToOctal(64));           // "100"
console.log(octalToDecimal("100"));        // 64
console.log(decodePermissions(755));       // "rwx r-x r-x"

Python Implementation

# Convert decimal to octal
def decimal_to_octal(num):
    return oct(num)[2:]  # Remove '0o' prefix

# Convert octal to decimal
def octal_to_decimal(octal_str):
    return int(octal_str, 8)

# File permission utilities
def permission_to_string(mode):
    """Convert octal permission to string format"""
    permissions = {
        '0': '---', '1': '--x', '2': '-w-', '3': '-wx',
        '4': 'r--', '5': 'r-x', '6': 'rw-', '7': 'rwx'
    }
    
    return ''.join(permissions[digit] for digit in str(mode))

# Examples
print(decimal_to_octal(255))           # "377"
print(octal_to_decimal("377"))         # 255
print(permission_to_string("755"))     # "rwxr-xr-x"
