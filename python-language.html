Language Overview

Python is a versatile, high-level programming language that emphasizes code readability and simplicity. Created by Guido van Rossum in 1991, Python has become one of the most popular programming languages due to its clean syntax, extensive standard library, and wide range of applications from web development to artificial intelligence.

üìö Basic Syntax

Hello World and Basic Structure

# This is a comment
print("Hello, World!")

# Variables and data types
name = "Python"           # String
age = 30                  # Integer
height = 1.75            # Float
is_awesome = True        # Boolean

# String formatting
print(f"Hello, {name}! You are {age} years old.")

# Basic operations
result = 10 + 5 * 2      # Result: 20 (follows order of operations)

üèóÔ∏è Data Structures

Lists

# Creating and manipulating lists
fruits = ["apple", "banana", "orange"]
fruits.append("grape")        # Add item
fruits.remove("banana")       # Remove item
print(fruits[0])              # Access first item
print(len(fruits))            # Get length

# List comprehension
squares = [x**2 for x in range(5)]
print(squares)  # [0, 1, 4, 9, 16]

Dictionaries

# Key-value pairs
person = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

# Accessing values
print(person["name"])         # John
print(person.get("age"))      # 30

# Adding/updating
person["job"] = "Developer"
person["age"] = 31

Tuples

# Immutable sequences
coordinates = (10, 20)
x, y = coordinates            # Unpacking

# Tuple methods
print(coordinates.count(10))  # Count occurrences
print(coordinates.index(20))  # Find index

Sets

# Unique, unordered collections
numbers = {1, 2, 3, 3, 4}    # Duplicates removed
numbers.add(5)                # Add element
numbers.remove(1)             # Remove element

# Set operations
set1 = {1, 2, 3}
set2 = {3, 4, 5}
union = set1 | set2           # Union
intersection = set1 & set2    # Intersection

üîÑ Control Flow

Conditional Statements

age = 18

if age >= 18:
    print("You are an adult")
elif age >= 13:
    print("You are a teenager")
else:
    print("You are a child")

# Ternary operator
status = "adult" if age >= 18 else "minor"

Loops

# For loop
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# While loop
count = 0
while count < 3:
    print(f"Count: {count}")
    count += 1

# Loop with enumerate
fruits = ["apple", "banana", "orange"]
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")

Functions

# Basic function
def greet(name):
    return f"Hello, {name}!"

# Function with default parameters
def power(base, exponent=2):
    return base ** exponent

# Lambda function
square = lambda x: x**2
print(square(5))  # 25

üèõÔ∏è Object-Oriented Programming

Classes and Objects

class Dog:
    def __init__(self, name, breed):
        self.name = name
        self.breed = breed
    
    def bark(self):
        return f"{self.name} says Woof!"
    
    def get_info(self):
        return f"{self.name} is a {self.breed}"

# Creating objects
my_dog = Dog("Buddy", "Golden Retriever")
print(my_dog.bark())         # Buddy says Woof!
print(my_dog.get_info())     # Buddy is a Golden Retriever

# Inheritance
class Puppy(Dog):
    def __init__(self, name, breed, age):
        super().__init__(name, breed)
        self.age = age
    
    def play(self):
        return f"{self.name} is playing at age {self.age}"

üîß File Handling

Reading and Writing Files

# Reading a file
with open('file.txt', 'r') as file:
    content = file.read()
    print(content)

# Writing to a file
with open('output.txt', 'w') as file:
    file.write("Hello, Python!")

# Reading line by line
with open('file.txt', 'r') as file:
    for line in file:
        print(line.strip())

# Working with CSV
import csv
with open('data.csv', 'r') as file:
    reader = csv.DictReader(file)
    for row in reader:
        print(row['name'], row['age'])

üêõ Error Handling

Try-Except Blocks

try:
    number = int(input("Enter a number: "))
    result = 10 / number
    print(f"Result: {result}")
except ValueError:
    print("Please enter a valid number")
except ZeroDivisionError:
    print("Cannot divide by zero")
except Exception as e:
    print(f"An error occurred: {e}")
finally:
    print("This always executes")

# Custom exceptions
class CustomError(Exception):
    pass

def validate_age(age):
    if age < 0:
        raise CustomError("Age cannot be negative")
    return age

üöÄ Best Practices

1. Follow PEP 8

Use consistent naming conventions, proper indentation, and line length limits.

2. Use Virtual Environments

Create isolated environments for different projects to manage dependencies.

3. Write Docstrings

Document your functions and classes with clear docstrings.

4. Use List Comprehensions

Prefer list comprehensions over traditional for loops when appropriate.

5. Handle Exceptions Properly

Use specific exception types and avoid bare except clauses.
