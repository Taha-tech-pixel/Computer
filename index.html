ASCII (American Standard Code for Information Interchange)

A 7-bit character encoding scheme that represents 128 characters including English letters, digits, punctuation marks, and control characters. It's the foundation of modern text encoding.

7-bit 128 Characters English Only

Key Features:

  • Uses 7 bits (0-127)
  • Standard for English text
  • Includes control characters (0-31)
  • Printable characters (32-126)
  • Delete character (127)
// ASCII Examples
'A' = 65 (01000001)
'a' = 97 (01100001)
'0' = 48 (00110000)
' ' = 32 (00100000)
'\n' = 10 (00001010)
Learn More
üåç

Unicode

A universal character encoding standard that supports virtually all writing systems and languages in the world. It provides a unique number for every character across different platforms and languages.

Universal Multi-language Extensible

Key Features:

  • Supports over 1.1 million characters
  • Backward compatible with ASCII
  • Multiple encoding forms (UTF-8, UTF-16, UTF-32)
  • Includes emojis and symbols
  • International standard (ISO/IEC 10646)

Object-Oriented Programming

Master the fundamentals of OOP

Encapsulation

Bundling data and methods that operate on that data within a single unit.

class BankAccount {
    private balance: number;
    
    constructor(initialBalance: number) {
        this.balance = initialBalance;
    }
    
    public getBalance(): number {
        return this.balance;
    }
    
    public deposit(amount: number): void {
        if (amount > 0) {
            this.balance += amount;
        }
    }
}

Inheritance

Creating new classes that are built upon existing classes.

class Animal {
    protected name: string;
    
    constructor(name: string) {
        this.name = name;
    }
    
    public makeSound(): void {
        console.log('Some sound');
    }
}

class Dog extends Animal {
    public makeSound(): void {
        console.log('Woof!');
    }
}

Polymorphism

Ability to present the same interface for different underlying forms.

interface Shape {
    calculateArea(): number;
}

class Circle implements Shape {
    constructor(private radius: number) {}
    
    calculateArea(): number {
        return Math.PI * this.radius ** 2;
    }
}

class Rectangle implements Shape {
    constructor(private width: number, private height: number) {}
    
    calculateArea(): number {
        return this.width * this.height;
    }
}
