Language Overview

JavaScript is a versatile, high-level programming language that powers the interactive web. Originally created to add interactivity to web pages, JavaScript has evolved into a full-featured programming language used for frontend and backend development, mobile apps, and even desktop applications.

Variables and Data Types

// Variable declarations
let name = "JavaScript";        // Block-scoped variable
const version = "ES2023";       // Constant
var oldWay = "Legacy";         // Function-scoped (avoid)

// Data types
let string = "Hello World";     // String
let number = 42;               // Number
let boolean = true;            // Boolean
let array = [1, 2, 3];        // Array
let object = {                 // Object
    key: "value",
    method: function() {
        return "Hello";
    }
};
let nothing = null;            // Null
let undefined;                 // Undefined

// Template literals
let greeting = `Hello, ${name}! Welcome to ${version}`;

üèóÔ∏è Functions and Scope

Function Declarations

// Function declaration
function greet(name) {
    return `Hello, ${name}!`;
}

// Function expression
const greetFunc = function(name) {
    return `Hello, ${name}!`;
};

// Arrow function
const greetArrow = (name) => `Hello, ${name}!`;

// Arrow function with multiple parameters
const add = (a, b) => a + b;

Closures and Scope

function createCounter() {
    let count = 0;
    
    return function() {
        count++;
        return count;
    };
}

const counter = createCounter();
console.log(counter()); // 1
console.log(counter()); // 2

// The inner function has access to 'count'

üîÑ Modern JavaScript Features

Destructuring

// Array destructuring
const [first, second, ...rest] = [1, 2, 3, 4, 5];
console.log(first);  // 1
console.log(rest);   // [3, 4, 5]

// Object destructuring
const { name, age, ...otherProps } = {
    name: "John",
    age: 30,
    city: "NYC",
    job: "Developer"
};

Spread and Rest

// Spread operator
const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5]; // [1, 2, 3, 4, 5]

const obj1 = { foo: 'bar' };
const obj2 = { ...obj1, baz: 'qux' };

// Rest parameters
function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}

Async/Await

async function fetchUser(id) {
    try {
        const response = await fetch(`/api/users/${id}`);
        const user = await response.json();
        return user;
    } catch (error) {
        console.error('Error:', error);
    }
}

// Using the async function
fetchUser(123).then(user => {
    console.log(user);
});

üöÄ Best Practices

1. Use Modern Syntax

Prefer const and let over var, use arrow functions, and implement destructuring.

2. Handle Errors Properly

Use try-catch blocks and implement proper error handling for async operations.

3. Avoid Global Variables

Use modules and proper scoping to avoid polluting the global namespace.

4. Write Clean Code

Follow consistent naming conventions and use meaningful variable names.

5. Use ES6+ Features

Leverage modern JavaScript features like classes, modules, and async/await.
