Hexadecimal Number System (Base-16)

The compact base-16 system using digits 0-9 and letters A-F

Base-16 Hex Programming Memory Addresses

Overview

What is Hexadecimal?

Hexadecimal is a base-16 number system that uses sixteen symbols: digits 0-9 and letters A-F. It's widely used in computing because it provides a more compact way to represent binary data, with each hexadecimal digit representing exactly four binary bits.

Why Hexadecimal?

Hex is popular in programming because it's much more readable than binary while maintaining a direct relationship with binary values. One hex digit = 4 bits, making conversion straightforward and hex numbers much shorter than their binary equivalents.

Common Uses

Hexadecimal is extensively used for memory addresses, color codes in web design, assembly language programming, debugging, and representing binary data in a human-readable format.

Programming with Hexadecimal

JavaScript Implementation

// Convert decimal to hexadecimal
function decimalToHex(num) {
    return num.toString(16).toUpperCase();
}

// Convert hexadecimal to decimal
function hexToDecimal(hexStr) {
    return parseInt(hexStr, 16);
}

// RGB to hex color
function rgbToHex(r, g, b) {
    return "#" + 
           r.toString(16).padStart(2, '0') +
           g.toString(16).padStart(2, '0') +
           b.toString(16).padStart(2, '0');
}

// Examples
console.log(decimalToHex(255));        // "FF"
console.log(hexToDecimal("FF"));       // 255
console.log(rgbToHex(255, 0, 0));      // "#ff0000"

Python Implementation

# Convert decimal to hexadecimal
def decimal_to_hex(num):
    return hex(num)[2:].upper()  # Remove '0x' prefix

# Convert hexadecimal to decimal
def hex_to_decimal(hex_str):
    return int(hex_str, 16)

# Color manipulation
def lighten_color(hex_color, factor=0.1):
    hex_color = hex_color.lstrip('#')
    
    # Convert to RGB
    r = int(hex_color[0:2], 16)
    g = int(hex_color[2:4], 16)
    b = int(hex_color[4:6], 16)
    
    # Lighten each component
    r = min(255, int(r + (255 - r) * factor))
    g = min(255, int(g + (255 - g) * factor))
    b = min(255, int(b + (255 - b) * factor))
    
    return f"#{r:02X}{g:02X}{b:02X}"
