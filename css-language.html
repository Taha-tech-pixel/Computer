The styling language used to describe the presentation of HTML documents

Language Overview

CSS is a style sheet language used for describing the presentation of a document written in HTML. It controls the layout, colors, fonts, and overall visual appearance of web pages. CSS separates content from presentation, making websites more maintainable and accessible.

First Released 1996
Latest Version CSS3
Paradigm Style Sheet
Typing None

ðŸ“š Basic Syntax

CSS Rule Structure

selector {
    property: value;
    another-property: value;
}

/* Example */
.button {
    background-color: #007bff;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
}

.button:hover {
    background-color: #0056b3;
}

ðŸŽ¯ CSS Selectors

Element Selectors

/* Selects all <p> elements */
p {
    color: blue;
}

/* Selects all <h1> elements */
h1 {
    font-size: 24px;
}

Class Selectors

/* Selects elements with class="highlight" */
.highlight {
    background-color: yellow;
}

/* Selects <p> elements with class="intro" */
p.intro {
    font-weight: bold;
}

ID Selectors

/* Selects element with id="header" */
#header {
    background-color: #333;
    color: white;
}

Pseudo-classes

/* Selects links on hover */
a:hover {
    text-decoration: underline;
}

/* Selects first child element */
li:first-child {
    font-weight: bold;
}

ðŸŽ¨ CSS Properties

Text Properties

  • color - Text color
  • font-family - Font type
  • font-size - Text size
  • font-weight - Text thickness
  • text-align - Text alignment
  • line-height - Line spacing

Box Model Properties

  • width - Element width
  • height - Element height
  • margin - Outer spacing
  • padding - Inner spacing
  • border - Border styling
  • box-sizing - Box model type

Layout Properties

  • display - Display type
  • position - Positioning method
  • float - Float positioning
  • clear - Clear floating
  • z-index - Stacking order
  • overflow - Content overflow

Background Properties

  • background-color - Background color
  • background-image - Background image
  • background-size - Image sizing
  • background-position - Image position
  • background-repeat - Image repetition
  • background-attachment - Image attachment

ðŸ“± Responsive CSS

Media Queries

/* Mobile devices */
@media (max-width: 768px) {
    .container {
        width: 100%;
        padding: 10px;
    }
    
    .nav-menu {
        flex-direction: column;
    }
}

/* Tablet devices */
@media (min-width: 769px) and (max-width: 1024px) {
    .container {
        width: 90%;
        max-width: 800px;
    }
}

/* Desktop devices */
@media (min-width: 1025px) {
    .container {
        width: 80%;
        max-width: 1200px;
    }
}

Flexbox Layout

.flex-container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    flex-wrap: wrap;
}

.flex-item {
    flex: 1;
    min-width: 200px;
}

CSS Grid

.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 20px;
    padding: 20px;
}

ðŸš€ Best Practices

1. Use Semantic Class Names

Choose descriptive class names that reflect the purpose, not the appearance.

2. Follow CSS Methodology

Use consistent naming conventions like BEM, SMACSS, or OOCSS.

3. Optimize for Performance

Minimize CSS file size and avoid overly specific selectors.

4. Mobile-First Approach

Start with mobile styles and enhance for larger screens.

5. Use CSS Variables

Implement CSS custom properties for consistent theming.
