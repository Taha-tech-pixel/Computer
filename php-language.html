Language Overview

PHP (Hypertext Preprocessor) is a widely-used open source general-purpose scripting language that is especially suited for web development and can be embedded into HTML. It's known for its simplicity, flexibility, and extensive library support.

Latest Version

Paradigm

Dynamic, Weak

Key Features

Server-Side

Executes on the server, generates HTML for clients

Cross-Platform

Runs on Windows, Linux, macOS, and Unix systems

Database Support

Built-in support for MySQL, PostgreSQL, SQLite

Frameworks

Laravel, Symfony, CodeIgniter, WordPress

Basic Syntax

<?php
// Variables
$name = "John";
$age = 30;
$isActive = true;

// Arrays
$fruits = ["apple", "banana", "orange"];
$person = [
    "name" => "John",
    "age" => 30,
    "city" => "New York"
];

// Functions
function greet($name, $age) {
    return "Hello $name, you are $age years old";
}

// Classes
class User {
    private $name;
    private $email;
    
    public function __construct($name, $email) {
        $this->name = $name;
        $this->email = $email;
    }
    
    public function getName() {
        return $this->name;
    }
}

// Usage
$user = new User("John", "john@example.com");
echo $user->getName();
?>
